## Instagram Post Performance Analysis

This project analyzes Instagram post performance to understand what
drives reach, engagement, and follower growth using Python and
exploratory data analysis (EDA).

### Objectives
- Analyze source-wise impressions (Home, Hashtags, Explore, Other)
- Calculate engagement and engagement rate
- Identify factors influencing profile visits and follows
- Understand relationships between key performance metrics

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
